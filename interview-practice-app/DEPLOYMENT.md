## 🚀 デプロイメント手順

### 1. 環境準備
```bash
# 依存関係インストール
npm install

# 環境変数設定
cp .env.example .env.local
# OPENAI_API_KEY、DATABASE_URL等を設定
```

### 2. データベース設定
```bash
# Prismaセットアップ
npm run db:generate
npm run db:migrate
npm run db:seed
```

### 3. ビルドとテスト
```bash
# TypeScriptチェック
npm run typecheck

# Lintチェック
npm run lint

# プロダクションビルド
npm run build
```

### 4. 本番デプロイ
```bash
# 本番サーバー起動
npm start
```
